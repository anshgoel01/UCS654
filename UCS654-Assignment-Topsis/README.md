# Topsis Assignment - UCS654

This project implements the Topsis method in three parts as required in the assignment.

## Part-I
Command line implementation of Topsis.
Takes CSV file, weights and impacts and generates result file.

## Part-II
Python package uploaded on PyPI.
Package Name: Topsis-Manmeet-102317039

Install using:
pip install Topsis-Manmeet-102317039

Run using:
topsis input.csv "weights" "impacts" output.csv

## Part-III
Flask based web service.
User uploads file, weights, impacts and email id.
Result CSV is sent through email.

## Technologies Used
- Python
- Pandas
- Numpy
- Flask
